# Bankcard_OCR
本代码简易的实现了银行卡识别的功能，通过深度学习（CTPN、Densenet、CTC）实现银行卡号的识别。

## 1、快速开始：
  本代码仅在Ubuntu下通过测试
  #### 1.1环境部署
  ```angular2
  sh setup.sh
  ```
   #### 1.2将测试图片放入demo/test_images目录，检测结果会保存到demo/test_result中
   #### 1.3执行demo文件下的demo.py
  ```angular2
  python demo.py
  ```
